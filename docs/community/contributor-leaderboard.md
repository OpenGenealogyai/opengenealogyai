## Badges

Contributors earn badges displayed on their GitHub profile (via README update):

| Badge | Criteria |
|-------|----------|
| First Record | First validated record merged |
| Collector | 1 collection adopted and completed |
| Archivist | 5 collections completed |
| Validator | 50 human reviews submitted |
| Privacy Guardian | Caught a living-person flag error in a PR |
| Standard Setter | Contributed to schema design or conflict protocol |

---

## How Counts Are Calculated

- **Records Extracted:** Count of `extracted_by: {contributor_id}` in merged raw-record files
- **Collections Adopted:** Collections where contributor is listed in `adopt-a-collection-tasks.md`
- **Assertions Validated:** Human review forms submitted and incorporated into confidence scores

All counts are computed from the git history and are immutable once merged.
